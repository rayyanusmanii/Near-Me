# Near Me

**Near Me** is a full-stack mapping web app with a React/Leaflet frontend and Node.js backend. 
It uses Google Gemini AI for semantic search and personalized route generation, and integrates real-time weather and traffic updates for an interactive user experience.

![Screenshot](assets/store_screenshot.jpg)  
*Feel free to check out the project [here](https://nrme.site)!*

---

## Features

- **Interactive Map:** Built with React and Leaflet to provide smooth map interactions and real-time updates.  
- **AI-Powered Search:** Uses Google Gemini AI to plan personalized multi-stop routes based on your preferences.
- **Real-Time Updates:** Fetches weather and traffic data through REST APIs for accurate, up-to-date information.  
- **Full-Stack Architecture:** Node.js backend handles data and AI prompts seamlessly.
